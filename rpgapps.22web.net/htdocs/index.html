App description

A collection of popular pocket games. There's a new game added to the pack, with every free update. So, the excitement never ceases to exist. Current games: 4+ and counting...


HANGMAN Hangman is a word guessing game. If the guessing player suggests a letter which occurs in the word, the letter is written it in all its correct positions. If the suggested letter does not occur in the word, one element of the hangman diagram is drawn as a tally mark. The game continues till the diagram is complete, or the player correctly guesses the word. With a vast wordlist this game gets really exciting.


SUDOKU Sudoku is a number-placement puzzle. The objective is to fill a 9×9 grid with digits so that each column, each row, and each of the nine 3×3 sub-grids that compose the grid contains all of the digits from 1 to 9. With about 100+ in-built puzzles, and with multiple solutions, this game is a perfect brain teaser.


JUMBLED

Jumbled is a game where you have to find the word whose letters have been mixed up. This game generates random words from an inbuilt list of about 14,481 commonly used words. This game is a very nice addition to the pack.


COWBULL (with update 1.1)

CowBull, also known as Bulls and Cows, is a code breaking game, which tests your mental ability and your presence of mind. You have to guess the secret number with the minimum number of guessings. As soon as you learn the basics, this game will you make you addicted to it!
